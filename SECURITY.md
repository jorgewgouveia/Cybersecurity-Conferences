# Security Policy

While this repository primarily contains public conference information, we still take security and integrity seriously.

## Reporting a Vulnerability or Sensitive Issue

- If you find incorrect, malicious, or sensitive content (e.g., non-public data unintentionally included), please open a **security-related issue** marked clearly, or contact the maintainers privately.
- Do not include sensitive personal information in public issues.

## Response

We aim to acknowledge reports within 7 days and remediate as soon as practical.
