# Contributing Guide

Thank you for helping improve Cybersecurity Conferences 2026!

## How to Add or Update a Conference

1. Identify the appropriate country folder under `conferences/` (e.g., `conferences/USA/README.md`).
2. Add a new table row with:
   - Event Name
   - Date
   - Venue
   - Website (official link)
   - Short Description
   - Country flag
3. If the country folder does not exist:
   - Create `conferences/<Country>/README.md`.
   - Add a table following the format in other country files.

## Pull Requests

- Keep PRs focused and small.
- Update only relevant files; avoid unrelated changes.
- Ensure links resolve and are official.
- Use the PR template checklists.

## Issues

- Use the provided issue templates for bugs and feature requests.
- Include specific file paths and entry names.

## Code of Conduct

By participating, you agree to the [Code of Conduct](CODE_OF_CONDUCT.md).
